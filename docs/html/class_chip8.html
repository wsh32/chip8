void Chip8::loadRom ( const char *  path)

Loads a ROM file into memory. The location in memory allocated to ROM data is from 0x200 to 0xE9F, which is defined by ROM_START and ROM_END in chip8.h. If the ROM file is bigger than the available space, it will throw an invalid_argument exception.

Parameters
path: Path to the ROM file. Use an absolute path or a path relative to the user's current directory.
Exceptions
invalid_argumentif the ROM file is greater than the allocated memory
